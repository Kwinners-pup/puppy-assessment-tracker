Temperament Assessment Tracker Help

Getting Started

  • Enter a YouTube Video ID (the part after "v=" in a YouTube URL) and click "Load Video"
  • Click "Add New Event" to create timestamps for behaviors you observe
  • Your data is automatically saved in your browser

Adding and Editing Events

  • Click on any event in the timeline to edit it
  • Enter a timestamp (MM:SS format), description, and tags
  • Click "Save Event" to save your changes
  • Click "Delete Event" to remove an event

Using Tags

  • Add tags to categorize behaviors (e.g., "social", "fearful", "confident")
  • Click on any tag in the timeline or filter bar to filter events by that tag
  • Use "Clear Filters" to show all events again

Importing and Exporting

  • "Import JSON" - Upload a previously saved assessment file
  • "Download JSON" - Save your assessment as a JSON file
  • "Export Text" - Create a plain text version of your assessment
  • "Save to Browser" - Manually save to your browser's storage
